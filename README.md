# Python Data Processing Tool

A simple, modular Python application for loading, cleaning,
and analyzing structured datasets.

## Features
- Load CSV and JSON data
- Basic data cleaning and validation
- Descriptive statistics
- Export of processed results

## Technologies
- Python 3
- pandas
- numpy

## How to Run
```bash
pip install -r requirements.txt
python src/main.py --input data/sample_data.csv
